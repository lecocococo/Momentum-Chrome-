# Momentum-Chrome-
Building Main function of Momentum(Google Chrome Extension)

이 프로젝트는 HTML,CSS,JS를 이용한 크롬 확장 프로그램인 Momentum을 구현해본 프로젝트입니다.

입력받은 데이터 저장은 브라우저의 Local Storage를 이용하였습니다.
